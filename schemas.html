🖼 GESTIONNAIRE DE SCHÉMAS

Ajoute des images/schémas aux termes cliquables du jeu

← Retour au jeu
📌 Comment ça marche : clique sur une carte pour importer une image (PNG, JPG, SVG, GIF). Elle s'affichera dans la fenêtre du terme cliquable (textes verts) directement dans le jeu. Les images sont stockées localement dans le navigateur.
💡 Conseil : utilise des schémas clairs, fond blanc ou transparent, résolution ≥ 400px de large. SVG recommandé pour la netteté sur écran tactile.
⚠️ Les données sont sauvegardées dans localStorage du navigateur — les images restent tant qu'on ne vide pas le cache.
