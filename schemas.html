🖼 GESTIONNAIRE DE SCHÉMAS

Ajoute des images/schémas aux termes cliquables du jeu

← Retour au jeu
📌 Comment ça marche : clique sur une carte pour importer une image (PNG, JPG, SVG, GIF). Elle s'affichera dans la fenêtre du terme cliquable directement dans le jeu.
💾 Sauvegarder : clique "Générer JSON" → copie ou télécharge le fichier → commit schemas_data.json dans ton dépôt GitHub → GitHub Pages déploie automatiquement !
🌐 Les images sont stockées dans le fichier schemas_data.json de ton repo GitHub — accessibles sur tous les appareils.
